Universal Architecture 普遍的アーキテクチャ

The C4 model,
applied to everything. C4モデルを、
あらゆるものへ。

Originally a software architecture framework, the C4 model — Context, Container, Component, Code — provides a universal language for structural thinking.

ソフトウェアアーキテクチャの設計手法として生まれたC4モデルは、コンテキスト・コンテナ・コンポーネント・コードの4層構造で、思考のあらゆる場面に応用できる普遍的な言語です。

At C4 Model Study Group®, we extend this lens to haiku poetry, literary analysis, music cognition, and AI-assisted academic writing.

C4モデル研究会®では、この視点を俳句・文学分析・音楽認知・AI学術論文執筆に拡張し、新しい知的生産の方法論を探求しています。

L1 · CONTEXT
World & Background
世界観・背景
Society, era, external actors, and the "why"
社会・時代・外部アクター・「なぜ」の問い
L2 · CONTAINER
Structure & Framework
章立て・枠組み
Logical containers: intro → theory → implementation → discussion
論理の器：序論→理論→実装→考察
L3 · COMPONENT
Arguments & Evidence
論点・データ
Claims, evidence, experimental data, citations
主張・根拠・実験データ・引用
L4 · CODE
Sentences & Connectives
文・接続詞
Connectives, tense consistency, citation numbers
接続詞・文末統一・引用番号
